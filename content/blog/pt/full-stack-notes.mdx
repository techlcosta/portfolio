# Construindo software como um laboratório contínuo

Desenvolver para a web sempre foi uma forma de transformar ideias em ferramentas reais. No dia a dia, gosto de conectar frontend, backend, automações e integrações para criar soluções que sejam simples de usar e bem estruturadas por dentro.

## O que me guia

Uma boa aplicação não nasce apenas de uma tecnologia específica. Ela combina clareza no problema, uma interface bem pensada, uma API confiável e uma base técnica que permita evoluir sem travar o produto.

## Tecnologias como repertório

Meu stack passa por React, Next.js, TypeScript, Node.js, Python e bancos de dados, mas o ponto mais importante é continuar aprendendo. Gosto de estudar novas ferramentas, testar abordagens diferentes e entender onde cada tecnologia pode gerar valor de verdade.

## Mercado financeiro como hobby técnico

O mercado financeiro também aparece nos meus estudos e projetos pessoais. MQL5, MetaTrader e TradingView entram como um campo interessante para experimentar automações, leitura de dados, integrações e formas de transformar estratégia em software.

## Próximos passos

Este blog será um espaço para registrar aprendizados, experimentos, decisões técnicas e pequenas descobertas feitas durante a construção de projetos full stack.
